Branding

Authentiq branding guidelines

Have a need for Authentiq's brand resources? Great! We have only a few guidelines we follow, and in turn ask you to follow as well. These guidelines were inspired by Bootstrap's Brand Assets, which has been inspired by MailChimp's ones.

Mark and logo

Mark

Use either the Authentiq mark or the standard logo (just Authentiq). It should always appear in Helvetica Neue Bold.
